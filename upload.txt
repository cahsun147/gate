**Langkah 3: Refaktor Root Layout & Global CSS**

1.  **File: `app/globals.css`**
    Hapus **semua isi** file ini. Terutama 3 baris `@tailwind`. Biarkan file ini kosong.

2.  **File: `app/layout.tsx`**
    Ganti total isinya dengan kode berikut untuk menginisialisasi Arwes, *sound*, dan *style* dasar.

    ```tsx
    // app/layout.tsx
    'use client'; // Provider Arwes memerlukan Client Component

    import React from 'react';
    import { Inter } from "next/font/google";

    // Hapus import "./globals.css" jika ada, atau biarkan jika filenya sudah kosong
    import "./globals.css"; 

    // Import provider Arwes
    import {
      ArwesProvider,
      SoundsProvider,
      StylesBaseline,
      Global
    } from '@arwes/react';

    // Import file suara (path ini akan valid setelah 'npm install')
    import audioClick from '@arwes/sounds/assets/sounds/click.mp3';
    import audioHover from '@arwes/sounds/assets/sounds/hover.mp3';
    import audioType from '@arwes/sounds/assets/sounds/type.mp3';

    const inter = Inter({ subsets: ["latin"] });

    // Definisikan audio untuk setiap aksi
    const players = {
      click: { sound: { src: [audioClick] } },
      hover: { sound: { src: [audioHover] } },
      type: { sound: { src: [audioType] }, loop: true }
    };

    export default function RootLayout({
      children,
    }: Readonly<{
      children: React.ReactNode;
    }>) {
      return (
        <html lang="en">
          <body className={inter.className}>
            {/* Ganti ThemeProvider Shadcn dengan ArwesProvider */}
            <ArwesProvider>
              <StylesBaseline /> {/* CSS Reset dari Arwes */}
              <Global /> {/* Style font dan dasar Arwes */}
              {/* Bungkus children dengan SoundsProvider */}
              <SoundsProvider players={players}>
                <main>{children}</main>
              </SoundsProvider>
            </ArwesProvider>
          </body>
        </html>
      );
    }
    ```

**Langkah 4: Aturan Refaktor Komponen & Halaman (Paradigma Baru)**

Terapkan aturan ini ke semua file di `app/` dan `components/`:

1.  **Tambahkan `'use client';`:** Tambahkan `'use client';` di baris paling atas setiap file halaman (`page.tsx`) dan komponen yang akan menggunakan komponen Arwes (karena Arwes menggunakan *hooks* dan *context*).
2.  **Hapus `className`:** Hapus SEMUA *props* `className` yang terkait dengan Tailwind (`p-4`, `grid`, `flex`, `text-lg`, `font-bold`, `bg-card`, dll.) dari setiap elemen JSX.
3.  **Ganti Layout:** Ganti *utility class* Tailwind dengan *prop* `style` atau komponen `@emotion/styled`.
      * *Contoh:* `className="grid grid-cols-4 gap-4"` menjadi `style={{ display: 'grid', gridTemplateColumns: 'repeat(4, 1fr)', gap: '1rem' }}`.
4.  **Bungkus Halaman dengan `<Animator>`:** Di setiap file `page.tsx` (misal, `app/dashboard/page.tsx`), bungkus seluruh konten utama halaman dengan komponen `<Animator>`. Ini penting untuk mengaktifkan animasi *staggered* (bergiliran).
5.  **Ganti Komponen Shadcn dengan Arwes:**
      * `Card`, `CardHeader`, `CardContent`, `CardTitle` -\> Ganti dengan satu komponen `<FrameLines as='div' padding={2} animator>...</FrameLines>`.
      * `Button` -\> Ganti dengan `<Button animator>...</Button>` dari `@arwes/react`. (Tombol ini akan otomatis berbunyi saat diklik/hover).
      * `h1`, `p`, `span`, `div` (untuk teks) -\> Ganti dengan `<Text as='h1' animator>`, `<Text as='p' animator>`, dll. dari `@arwes/react`.
      * `Table`, `TableRow`, `TableCell` -\> **Arwes tidak punya komponen `Table`**. Jaga struktur HTML `<table>` tetap utuh, tapi hapus semua `className`-nya. Bungkus seluruh `<table>` dengan `<FrameLines as='div' animator>...</FrameLines>`.
      * `Chart` (misal, `components/agent-dominance-chart.tsx`) -\> Jaga logika `recharts` tetap utuh. Hapus `className` dari *wrapper*-nya. Bungkus seluruh komponen chart dengan `<FrameLines as='div' animator padding={2}>...</FrameLines>`.
      * `app-sidebar.tsx` dan `nav-*.tsx` -\> Ini adalah refaktor besar. Ganti semua `div` dan `button` dengan komponen Arwes. Seluruh sidebar harus dibungkus `<FrameLines>`. Setiap item navigasi (`<Link>`) harus diubah menjadi `<Button as='a' href='...' animator>...</Button>`.
